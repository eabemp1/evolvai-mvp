# Roadmap

## MVP (now)
- Personal companion ownership and specialist routing.
- Reminders with browser + local native notifications.
- Mock marketplace: mint/list/buy/rent/train.
- Anonymized event pipeline and dataset snapshots.

## Phase 2
- Real authentication + per-user tenancy boundaries.
- Per-user partition for reminders/uploads/usage/memory at storage level.
- Test coverage expansion and CI gates.
- Structured observability and error budgets.

## Production
- Containerized deployment.
- Managed DB and migrations.
- Real queue/scheduler and background workers.
- Security hardening, rate limiting, abuse controls.
- Compliance and privacy controls for data governance.
